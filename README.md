#  Java Estudos

Repositório criado para armazenar exercícios e projetos desenvolvidos durante meus estudos em Java.

##  Conteúdos abordados

- Lógica de Programação
- Estruturas de decisão (if/else)
- Estruturas de repetição (while, for)
- Programação Orientada a Objetos (em andamento)
- Integração com Banco de Dados (em breve)

##  Objetivo

Construir uma base sólida em desenvolvimento back-end com Java e evoluir para projetos mais completos com integração a banco de dados.
